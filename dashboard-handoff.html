Current Prototype Behavior

  • Left menu is split into top navigation and a Favorites container. Original page remains untouched.
  • Dashboard list has an Ungrouped / Grouped switcher before All owners.
  • Ungrouped list has extra dashboards: Market Trends 2025, Customer Insights 2024, Revenue Forecasts Q3, Website Traffic Analysis 2024.
  • Collections column uses stack icon. One collection shows only stack icon; multiple collections show stack plus count such as +1; no collection shows no icon.
  • Grouped view is generated from the same dashboard collection membership model, not a separate static list.
  • Some dashboards intentionally belong to multiple collections and appear more than once in Grouped.
  • Grouped heart buttons toggle favorites and sync with Ungrouped rows plus the left dashboard list.
  • In temp, Favorites contains My objectives and My groups objectives, plus dashboard favorites. Each item shows its section icon at 16×16 with 70% opacity; on hover it changes to a drag handle.
  • In temp, Favorites are grouped into objective and dashboard sections with a separator only when both sections are present. Items can reorder only inside their own section; dropping into the other section returns the item. Holding an item over the upper or lower half of the other section for about one second moves its whole section.
  • In temp, Favorites navigation highlight is rendered on the actual rows of the relevant section, with no separate overlay layer.
  • In temp, hovering a grouped +1 collection badge highlights the related dashboard row if visible, or the related collapsed collection row if the dashboard is hidden.
  • In temp, clicking a collection badge in Ungrouped switches to Grouped, opens the relevant collection or collections, and highlights the matching dashboard rows with timed animation.
  • In temp, dragging an open collection carries its visible child dashboard rows as one bundle and uses an elevation state without rounded corners.
  • In temp, dragged open collection bundles keep a separator directly under the collection header row, before the first child dashboard row.
  • In temp, an open collection drags as a full bundle with visible dashboard rows in both the source projection and fixed-position drag preview. The drag preview is anchored to the pointer near the collection header, keeps the drag handle visible, and keeps elevation without rounded corners. The source projection stays at 50% opacity, has no elevation, and hides the drag handle.
  • In temp, dragging a grouped dashboard while holding Alt or Meta copies it into the target collection instead of moving it. A duplicate hint appears when the gesture is available, and copied rows are highlighted after drop.
  • In temp, grouped collection rows have Copy link and more-actions controls. Copy link switches to Link copied for 1.2 seconds, then returns to the normal or hover state.
  • In temp, grouped collection action controls use the same right action column as dashboard row controls, so the collection more-actions button aligns with dashboard kebab buttons.
  • In temp, selected collections are disabled in the collection assignment popover to avoid adding a dashboard to the same collection twice.
  • In temp, the Ungrouped / Grouped switcher now changes state instantly; the earlier full mode-switch animation is disabled.
  • In temp, related hover and tooltip state is force-cleared on pointer move away, scroll, click, blur, and visibility change to avoid stuck hover rows.
  • In temp, Grouped keeps its own header separator layer visible while collections are dragged or reordered; row separators still keep the same left offset as Ungrouped.
  • In temp, dashboard/objective preview snapshots are suppressed while the pointer is down or an item is being dragged; previews are hover-only.
  • Libertex workspace dropdown opens but has no selection behavior yet.

Design Rules To Keep

  • Live Oboard product is the primary source of truth for spacing, icons, states, and component behavior.
  • Do not invent icons. Use existing Oboard SVG symbols already in the page.
  • Collections are labels/tags, not owners or permission containers.
  • A dashboard can belong to multiple collections.
  • Collection rows should not show collection owner or visibility as collection-level properties.
  • Empty heart outline appears only on row hover. Filled heart is visible in default and hover states.
